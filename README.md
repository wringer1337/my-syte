# Висла Гласс — сайт с авторизацией, ролями, админкой и калькулятором

## Установка

1. Откройте терминал в папке проекта.
2. Выполните `npm install`.
3. Запустите сервер командой `npm start`.
4. Откройте в браузере http://localhost:3000.

## Доступы

- Админ: `admin@visla.ru` / `Admin123!`

## Структура

- `server.js` — сервер на Express и SQLite
- `public/` — фронтенд страницы
- `data/app.db` — база данных создаётся автоматически

## Возможности

- регистрация и вход
- роли: `user`, `manager`, `admin`
- админ-панель для управления пользователями
- калькулятор стоимости окон, дверей и стеклопакетов
- простой ИИ-помощник для консультаций
- личный кабинет пользователя
